# Foundry Management System - Application Overview

## Project Description
Create a comprehensive foundry management system with a dashboard, import from accounting and erp systems, job tracking, and multiple information management modules. The system should track jobs through various workflow stages (New, In Progress, Solidification/Casting, CAD Work, Waiting on CAM, Waiting on Sample, Completed) with color-coded status indicators throughout.  Some stages may require checklist completion prior to moving to the next step. 

## Core Architecture

### Layout
- Left sidebar navigation (collapsible with hamburger menu)
- Dark mode/light mode
- Sticky header with application title
- Responsive design with Tailwind CSS

Form Management Pattern
const openJobForm = (job = null) => {
  setEditingJob(job);
  if (job) {
    setFormData({ ...job }); // Pre-populate
  } else {
    setFormData({ /* empty fields */ });
  }
  setShowJobForm(true);
};

const closeJobForm = () => {
  setShowJobForm(false);
  setEditingJob(null);
};
